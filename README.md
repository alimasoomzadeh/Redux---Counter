# React, Redux

An experimental project (Redux)

## Getting Started  
`npm start`

## Prerequisites    
- Node.js
- npm

## Installing

#### Node.js
Download the node.js source code or a pre-Built installer for Your platform

#### npm
`npm install -g npm`  

#### redux
`npm install redux`    

#### react-redux
`npm install react-redux`   

  
## Business 

* counter-increment.
* counter-decrement.
